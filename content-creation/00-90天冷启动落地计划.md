# 宝宝的小红书0-1000粉冷启动落地计划（90天）

> **版本**: v1.0（经过4轮反思+2026年最新算法调研）
> **核心策略**: 前15篇极致垂直「居家调酒+酒类推荐」，搜索流量优先，AI提效，真实感第一
> **时间投入**: 前2周每周~5小时，之后每周~4小时
> **目标**: 90天1000粉，建立账号标签，跑出稳定内容模板

### Day 2：合规红线（酒类内容必看，非常重要！）
❌ 绝对不能出现的内容：
- 不要出现"喝酒能消愁/解压/解闷"等诱导性话术
- 不要出现未成年人（哪怕猫穿衣服都不行，更不要有小孩入镜）
- 不要说"千杯不醉""解酒""护肝"等医疗相关词汇
- 不要出现酗酒、拼酒、一口闷等画面
- 不要提"买酒加我""私我发链接"等导流话术（蒲公英之前不要导流）
- 笔记里不要出现微信/二维码/淘宝链接

✅ 安全话术模板：
- "下班回家小酌一杯，放松一下"
- "成年人的快乐水"
- "个人口味分享，理性饮酒"
- 最后可以加一句："未满18周岁禁止饮酒"（加了更安全）

## 🚀 第二阶段：标签建立期（Day 4-30，前15篇）

### 核心原则
1. **只发和酒直接相关的内容**：不要发猫、不要发健身、不要发工作感悟——前15篇就钉死"居家调酒/酒类推荐"标签
2. **一周3更**：周二、周四、周六晚8-10点发布（这个时间点用户活跃度高）
3. **图文为主**：前15篇全部做图文，不要做视频，单篇耗时控制在90分钟内
4. **每篇必须带关键词**：布局"居家调酒""适合女生喝的酒""金酒推荐""微醺"等搜索词

### 单篇图文创作SOP（90分钟标准流程）
| 步骤 | 时间 | 做什么 | 怎么用AI |
|------|------|--------|----------|
| 1. 选选题+拍照片 | 30分钟 | 在家调酒的时候顺便拍3-5张照片：原料图、制作过程、成品图、酒的特写 | - |
| 2. AI写初稿 | 10分钟 | 把选题、照片描述、你的个人风格输入AI，让它写文案初稿 | 用我给你的prompt |
| 3. 修改文案+写标题 | 20分钟 | 把AI写的文案改成你自己的语气，写3个标题选一个 | AI给你10个标题候选 |
| 4. 修图 | 20分钟 | 用醒图调个色（暖色调，有氛围感），加一点文字贴纸 | - |
| 5. 加话题+发布 | 10分钟 | 加5-8个相关话题，检查合规，发布 | - |

### AI写文案Prompt（直接复制用）
```
你是一个小红书酒类博主，人设是：
"8年数据产品经理，ESFP性格，真实不装，懂酒但不端着，像朋友一样分享"

请帮我写一篇小红书图文笔记，要求：
1. 主题：[这里填具体选题，比如：在家3分钟调一杯好喝的金汤力]
2. 语气：口语化，像和朋友聊天，不要太官方，有个人真实感受
3. 结构：
   - 开头第一句抓眼球（比如"下班回家累成狗，3分钟调杯酒续命"）
   - 中间分点讲：需要什么原料、步骤、小tips、口感感受
   - 结尾互动（比如"你们下班最爱喝什么？评论区告诉我"）
4. 字数：300-500字
5. 最后加一句"未满18周岁禁止饮酒，理性饮酒"
6. 给我10个标题选项，要包含关键词，有点击欲

我的个人偏好：
- 金酒喜欢诺迪斯、猴王47、添加利、亨利爵士
- 不喜欢太专业的术语，要接地气
- 真实分享，不硬推
```

### 发布后运营（发布后2小时黄金窗口）
1. 发布后10分钟内，自己先用小号/朋友号给第一条评论（比如"有没有同款下班就想喝一口的🙋"）
2. 有人评论**必须在1小时内回复**，而且要回复得长一点（不要只回"谢谢"，要聊两句，长评论权重高）
3. 不要刷赞刷评论！平台能检测到，会降权。找3-5个真朋友帮忙正常互动就行

## 📊 每周数据复盘SOP（每周日花30分钟）
用你数据产品经理的优势，每周做一次复盘：

| 指标 | 合格线 | 优秀线 | 怎么优化 |
|------|--------|--------|----------|
| 小眼睛（曝光） | ≥500 | ≥1500 | 封面/标题不行就换封面标题 |
| 点击率（小眼睛→点赞） | ≥3% | ≥8% | 封面不够吸引人，或者标题不对 |
| 收藏率 | ≥2% | ≥5% | 内容不够实用，增加干货 |
| 评论率 | ≥1% | ≥3% | 结尾加互动提问 |
| 转粉率 | ≥0.5% | ≥2% | 主页简介/内容垂直度不够 |

每周选数据最好的1篇，分析它为什么好，下周多做同类型内容；
选数据最差的1篇，分析问题，下次避开。

## 🔄 第三、四阶段（Day 31-90）
等前30天跑完，我们再根据实际数据调整计划。核心方向：
- 跑出1-2个爆款内容模板，复制放大
- 粉丝到500后，可以开始接酒吧/酒商的免费置换
- 粉丝到1000后，开通蒲公英，接付费广告
- 慢慢加入成都探店、个人生活内容，丰满人设

## ⚠️ 心态建设（最重要）
1. **接受前10篇数据都不好**：这非常正常，新号都有冷启动期，不要发了3篇没流量就放弃
2. **不要追求完美**：拍得不好看、写得有错别字都没关系，真实比完美重要
3. **不要和别人比**：有人一篇爆涨1万粉，有人3个月才到1000粉，每个人节奏不一样，坚持更重要
4. **这是你自己的生活记录**：哪怕最后没涨粉，你记录了自己的喝酒日常，认识了几个同好，也不亏
