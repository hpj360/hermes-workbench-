# Liquid Glass 典型组合模式

## 模式 1：悬浮工具栏

```
┌─────────────────────────────────┐
│  [主页]  [搜索]  [消息]  [我]    │  <- GlassNavigation (top)
├─────────────────────────────────┤
│                                 │
│   主内容区域（实色背景）         │
│                                 │
└─────────────────────────────────┘
```

**规则**：
- 工具栏高度 48-64px
- 主内容背景必须是色块/图片（不能是纯色，否则玻璃无效）
- 滚动时工具栏背景 alpha 从 0 升到 0.6

## 模式 2：侧边抽屉

```
┌──────────┬──────────────────────┐
│ [头像]   │                      │
│ [菜单]   │   主内容             │
│ [设置]   │                      │
│ ...      │                      │
└──────────┘
   ↑
   侧边栏 280-320px 宽
   GlassPanel blur=30 alpha=0.7
```

**规则**：
- 主内容区背景要有变化（图片/渐变）
- 抽屉打开时背后内容变暗（overlay alpha 0.3）

## 模式 3：Modal 弹窗

```
┌─────────────────────────────────┐
│                                 │
│      ┌───────────────┐         │
│      │   标题        │         │
│      │   内容        │ <- GlassPanel blur=50 alpha=0.8
│      │   [确定][取消]│         │
│      └───────────────┘         │
│                                 │
└─────────────────────────────────┘
```

**规则**：
- 背后 overlay 必须有（不能直接 GlassModal）
- Modal 居中，最大宽度 480px
- 入场动画 spring damping=0.85

## 模式 4：通知横幅

```
┌─────────────────────────────────┐
│ [图标] 标题文字        [关闭]   │  <- 顶部横幅，blur=20 alpha=0.7
└─────────────────────────────────┘
```

**规则**：
- 自动消失 4-6 秒
- 最多同时 1 个

## 模式 5：图片浏览器（推荐降级）

```
┌─────────────────────────────────┐
│  [关闭]                          │
│                                 │
│       [大图]                     │
│                                 │
│  [缩略图1] [缩略图2] ...        │  <- 底部控制栏，blur=24
└─────────────────────────────────┘
```

**注意**：大图模式下不推荐用玻璃（会失真），用半实色 overlay。

## 模式 6：浮动操作按钮（FAB）

```
                              ┌───┐
                              │ + │  <- GlassButton blur=20 alpha=0.7
                              └───┘
```

**规则**：
- 距屏幕边缘 16-24px
- 直径 56px
- press 时 alpha 升到 0.9
